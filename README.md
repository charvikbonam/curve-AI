# Curve AI

Intelligent chat assistant powered by Groq API with persistent conversations.

## Features
- Real-time AI chat
- Conversation history
- Dark/Light mode
- Mobile responsive

## Setup
1. Get Groq API key from console.groq.com
2. Run `export GROQ_API_KEY="your-key"`
3. Run `python3 app.py`
4. Open http://localhost:5000
